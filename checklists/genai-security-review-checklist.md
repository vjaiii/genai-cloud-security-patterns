# GenAI Security Review Checklist

## Identity

• Who can access the AI service  
• Service account permissions  
• API authentication mechanism  

## Data protection

• Is sensitive data used in prompts  
• Are retrieval sources protected  
• Encryption of data at rest  

## Prompt handling

• Prompt injection mitigation  
• Input validation  
• Prompt logging  

## Model access

• Is the model endpoint private  
• Are API tokens rotated  
• Access control enforced  

## Monitoring

• Prompt logs captured  
• API usage metrics  
• Alerting for abnormal usage
